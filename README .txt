                          Microservicio

1. Descripción:

Este proyecto es un Microservicio elaborado con Node.js, Express y PostgreSQL
se realizó con el fin de realizar el siguiente requerimiento: Teniendo en cuenta 
el siguiente modelo elabore un micro servicio para la tabla “empleado” el cual sea 
capaz de insertar, actualizar, borrar y consultar (CRUD) información utilizando él 
en torno de Node.js

2. Requisitos del sistema:

>Tener instalada una versión de Node js
>Ejecutar la BD de forma local en un gestor de base de datos de PosgreSql
>Abrir la aplicación en un editor de código, puede utilizar Visual Code 
>Ejecutar el comando: npm run dev o node src/index.js para compilar el proyecto
>El proyecto se puede consultar en el navegador http://localhost:1234/Consultar para verificar que este ejecutandose de forma correcta
>El Servicio se pude consultar en Postman
